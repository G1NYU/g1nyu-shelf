# G1NYU Shelf

A neo-otaku personal media shelf for manga, manhwa, music, and books.

## Stack
- Next.js 14
- Tailwind CSS
- Framer Motion
- AniList GraphQL API
- Last.fm API
- Open Library API

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local`:
```bash
LASTFM_API_KEY=your_lastfm_api_key
ANILIST_USERNAME=your_anilist_username
LASTFM_USERNAME=your_lastfm_username
OPENLIBRARY_READING_LIST=optional
```

3. Run locally:
```bash
npm run dev
```

## Deploy
Deploy to Vercel and add env vars in the Vercel dashboard.
